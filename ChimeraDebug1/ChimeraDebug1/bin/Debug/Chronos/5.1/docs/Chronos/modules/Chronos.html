Module Chronos

Chronos is a Lua framework for performing system tests using command sets through DeviceComm. 

 require "Chronos"
 
 local deviceComm
 
 local settings =
 {
   Port = "COM3",
   Baud = 38400,
   ReadLoopback = "true",
   AssertRTS = "true"
 }
 
 TestSuite "My Excellent Test Suite"
 {
   Setup = function()
     deviceComm = DeviceComm.new( "MyCommandset.xml" )
     deviceComm:Initialize( "Serial", settings )
   end,
 
   Teardown = function()
     deviceComm:Do( "AllDone" )
   end,
 
   EachSetup = function()
     deviceComm:Do( "Unlock" )
   end,
 
   TestCase "My Radical Test Case"
   {
     Setup = function()
       deviceComm:Do( "LetMeMessWithStuff" )
     end,
 
     function()
       local result = deviceComm:Do( "MessingWithTheMirror", { Param1 = 0xdeadbeef } )
       CheckEqual( 0x05, result.Out.ReturnValue, "Bad Things happened!" )
     end
   },
 
   TestCase "My Tubular Test Case"
   {
     Teardown = function()
       deviceComm:Do( "ResetValues" )
     end,
 
     Test = function()
       local result = deviceComm:Do( "ChangeSomeValues", { Value1 = 0x11, Value2 = 0x22, Value3 = 0x33 } )
       CheckSequenceHex( { 0x11, 0x22, 0x33 }, result.Out.SetValues )
       Check( result.Out.AwesomeFlag, StartHex() .. "Awesome flag was not set," ..
         " so I'm going to print some stuff in hex: " .. 123 .. " and " .. 45 .. EndHex() ..
         " but this is in decimal: " .. 67 )
     end
   }
 }

Functions

GenerateReport (results, templatePath)
Generates a report and returns it as a string. Report templates will have access to all of the elements of the test results, as well as the Chronos module and anything in the global table.

Parameters:

  • results: A TestRunResults table returned by a call to Chronos.Run().
  • templatePath: The path to the template.

Return value:

    The body of the generated template as a string
GetActiveRecorder ()
Gets the active test recorder that an action should use, preferring a Chronos.TestCaseRecorder to a Chronos.TestSuiteRecorder.

Return value:

    The active test recorder, or nil if there is none.
GetUserName ()
Provides a safe way to get the username of the shell running the script. Most useful in reports.

Return value:

    The username of the current user, or "User" if no information is available.
RenderReport (results, templatePath, outputPath)
Renders a report to a file.

Parameters:

  • results: A TestRunResults table returned by a call to Chronos.Run().
  • templatePath: The path to the template.
  • outputPath: The path to the desired output location.
