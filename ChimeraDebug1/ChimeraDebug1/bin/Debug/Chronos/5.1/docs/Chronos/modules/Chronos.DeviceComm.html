Module Chronos.DeviceComm

DeviceComm Modifications. Chronos hooks into certain DeviceComm facilities to provide more information in reports and to control error handling.
This module is not activated until you require "gScripted".

DeviceComm
DeviceComm modifications

Fields:

  • Do: A wrapper around DeviceComm:Do(); if the command fails, the current test case fails.
  • Write: A wrapper around DeviceComm:Write(); if the command fails, the current test case fails.
  • Read: A wrapper around DeviceComm:Read(); if the command fails, the current test case fails.
  • FailDo: A wrapper around DeviceComm:Do() that expects the command to fail; if the command succeeds, the current test case fails.
  • FailWrite: A wrapper around DeviceComm:Write() that expects the command to fail; if the command succeeds, the current test case fails.
  • FailRead: A wrapper around DeviceComm:Read() that expects the command to fail; if the command succeeds, the current test case fails.
