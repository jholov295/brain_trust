Chronos Chronos is a Lua framework for performing system tests using command sets through DeviceComm.
Chronos.Action Represents an action taken during test execution.
Chronos.Actions Actions and failures are recorded for each test case and test suite as you run checks.
Chronos.Actions.Checks Functions for validating conditions during test execution.
Chronos.Actions.Interactive Functions that require input in order for the test to continue.
Chronos.Actions.MakeCheck Utility for generating check functions according to certain conventions.
Chronos.Actions.Traceability Functions for injecting information into test cases for reporting purposes.
Chronos.Actions.Utilities Miscellaneous functions.
Chronos.Auto A typical usage of AutoChronos looks like this: 
 require "Chronos"
 require "Chronos.Auto"
 require "gScripted"
 
 local deviceComm
 
 TestSuite "Auto-Generated Test Cases"
 {
   	Auto "commandsets/Audi_KWP.xml"
   	{
   		{ Pattern = "Coding Block .*", Reqs = { "SR-123" } },
   		"ALS Return Pulse Width Values",
   		{   			Name = "Aim Calibration Values",
   			Params =
   			{
   				["First Row"] = { Valid = { 6, 10, 55 }, Invalid = { 1, 99 } },
   				["First Column"] = { Valid = { 50 }, Invalid = { 107 } },
   				["Retrieve_Stored_Error"] = { Valid = { 1 } }
   			},
   			Verify = true
   		}
 
   		Setup = function()
   			deviceComm = DeviceComm.new( "commandsets/Audi_KWP.xml" )
   		end,
 
   		Test = function( command )
   			local results = AutoRun( deviceComm, command )
   			AutoCheck( deviceComm, results )
   		end
   	}
 }
Important Notes
  • You can pass in the regular TestCase functions, Setup, Teardown, and Test.
  • The Test function here is different in that it takes an argument.
