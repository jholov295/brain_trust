Module Chronos.TestCase

Represents a test case. 

 testCase = Chronos.TestCase.new( "MyTestCase" )
 testCase.Test = function() end
 testCase.Setup = function() end
 testCase.Teardown = function() end

Tables

Chronos.TestCase
Contails details about the test case.

Fields:

  • Anonymous: True if the test case is anonymous.
  • Name: The name of the test case. (Read-only)
  • Verison: The version of the test case.
  • Test: The test function. Does the actual execution of the test.
  • Setup: The setup function. Called before the Test function.
  • Teardown: The teardown function. Called after the Test function. This is called even if the Test fuction fails.
