Module Chronos.TestSuiteResults

Contains information about a test suite execution.

Chronos.TestSuiteResults

Tables

Chronos.TestSuiteResults


Fields:

  • StartTime: The start time of the test run.
  • EndTime: The end time of the test run.
  • Time: The total time of the test run.
  • Anonymous: True if the test suite is anonymous.
  • Name: The name of the test suite.
  • Version: The version of the test suite.
  • LineNumber: The number of the line on which the test suite was defined.
  • SourceFile: The file in which the test suite was defined.
  • Status: "Passed" "Pending" or "Failed"
  • Passed: True if the suite passed.
  • Pending: True if the suite is pending.
  • Failed: True if the suite failed.
  • Steps: Contains both the actions and failures of the suite's execution, in order.
  • Steps.Setup: Contains steps from Setup.
  • Steps.Idle: Contains steps from Idle.
  • Steps.Teardown: Contains steps from Teardown.
  • Actions: Contains actions of the suite's executed actions.
  • Actions.Setup: Contains actions from Setup.
  • Actions.Idle: Contains actions from Idle.
  • Actions.Teardown: Contains actions from Teardown.
  • Failures: Contains failures of the suite's failed actions.
  • Failures.Setup: Contains failures from Setup.
  • Failures.Idle: Contains failures from Idle.
  • Failures.Teardown: Contains failures from Teardown.
  • TestCases: An array of Chronos.TestCaseResults. (Also indexed by name: TestCases["My Test Case"])
  • TestCases.Passed: An array of Chronos.TestCaseResults whose status is Passed.
  • TestCases.Pending: An array of Chronos.TestCaseResults whose status is Pending.
  • TestCases.Failed: An array of Chronos.TestCaseResults whose status is Failed.

Functions

Chronos.TestSuiteResults:AddAction (state, action)
Adds an action to the Actions and Actions[ state ] arrays. Will mark the results as passed if they would otherwise remain pending.

Parameters:

  • state: The state during which the action occurred.
  • action: The action.
Chronos.TestSuiteResults:AddFailure (state, failure)
Adds a failure to the Failures and Failures[ state ] arrays. Will mark the results as failed.

Parameters:

  • state: The state during which the failure occurred.
  • failure: The failure.
new (testSuite, saveResults)
Creates a new Chronos.TestSuiteResults.

Parameters:

  • testSuite: The test suite.
  • saveResults: {boolean} if true save the results to the filesystem as it runs.

Return value:

    A new Chronos.TestSuiteResults.
