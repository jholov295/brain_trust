Module cllib.String

Helper functions that can be used to help manipulate strings.

Author:

Ryan Pusztai

Release: 1.12 <01/09/2009>

Note:

    This module is part of cllib (Common Lua Library). You will need extra files to use these functions. See 'Usage:' section for more information.

Usage:

    To use these scripts all you need to do is require() at the top of the file.
    This is an example of using an included module: 
    require( "ModuleName" )
    This is an example of calling a function in that module: 
    ModuleName.CoolFunction( parameter1, parameter2 )

FormatMessage (msg)
Formats a string into ASCII, hex and decimal representaions.

Parameters:

  • msg: String that contains the message that needs to be converted.

Return value:

    String that is formated as hex (ASCII) [decimal].
ToAscii (val, separator)
Formats any type into an ASCII string. Be aware that only numbers that are less then 256 are allowed. This only applies to numbers, not any other data types.

Parameters:

  • val: Value to be converted. It can be a table and it will build the elements into a single string.
  • separator: [OPT] String to use to denote element separation. Defaults to "".

Return value:

    String that is formated as ASCII.
ToDec (val, separator, isSigned)
Formats any value into a decimal representaion. If it is a table of values each value is converted to decimal and then placed in a buffer of comma separated values. e.g. "123, 456, 78, 90" Be aware that if you have signed numbers it will treat every value in the table as signed.

Parameters:

  • val: Value to be converted. This can be a table and then a string of comma separated values will be returned.
  • separator: [OPT] String that represents what separater to use if a table is passed into the function. Defaults to ", ".
  • isSigned: [OPT] Boolean that if true converts the number as a signed value, else it treats it as an unsigned value. For a table of values it will treat all of them as signed if this is true, else all as unsigned.

Return value:

    If not a table it returns the val as a decimal representation. If a table it returns it as a string of comma separated values that are converted to decimal.

    Note:

      You should use 'tonumber()' from Lua if you don't need the dynamic handling of tables.
ToEscapedHex (s)
Formats a string with C-style hex escapes (\x) for every two characters.

Parameters:

  • s: String or single element table that contains the message that needs to be converted.

Return value:

    String that is formated with C-style hex escapes (\x).

    Note:

      This will truncate the last byte if it doesn't have 2 characters. For example: String.ToEscapedHex( "7FABCDE" ) would return "\x7F\xAB\xCD" Notice the missing 'E'.
ToHex (val, separator, useHexPrefix)
Formats any type into a string hex representaion.

Parameters:

  • val: Value to be converted. It can be a table and it will build the elements into a single string.
  • separator: [OPT] String to use to denote element separation. Defaults to "".
  • useHexPrefix: [OPT] Boolean that should be true if you want the '0x' to be prefixed to the returned string. Defaults to true. e.g. 0xAB0438

Return value:

    String that is formated as hex.
Trim (s)
Trims white space from s.

Parameters:

  • s: String to trim the whitespace from.

Return value:

    String with all the whitespace removed from the begining and end of s.
