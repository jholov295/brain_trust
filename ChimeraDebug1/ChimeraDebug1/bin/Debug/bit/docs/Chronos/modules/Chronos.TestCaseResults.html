Module Chronos.TestCaseResults

Contains information about a test case execution.

TestCaseResults

Functions

Tables

TestCaseResults


Fields:

  • Anonymous: True if the test case is anonymous.
  • Name: The name of the test case.
  • Version: The version of the test case.
  • LineNumber: The number of the line on which the test case was defined.
  • SourceFile: The file in which the test case was defined.
  • Status: "Passed" "Pending" or "Failed"
  • Passed: True if the test passed.
  • Pending: True if the test is pending.
  • Failed: True if the test failed.
  • Steps: Contains both the actions and failures of the test's execution, in order.
  • Steps.Setup: Contains steps from Setup.
  • Steps.EachSetup: Contains steps from EachSetup.
  • Steps.Test: Contains steps from Test.
  • Steps.EachTeardown: Contains steps from EachTeardown.
  • Steps.Teardown: Contains steps from Teardown.
  • Actions: Contains actions of the test's executed actions.
  • Actions.Setup: Contains actions from Setup.
  • Actions.EachSetup: Contains actions from EachSetup.
  • Actions.Test: Contains actions from Test.
  • Actions.EachTeardown: Contains actions from EachTeardown.
  • Actions.Teardown: Contains actions from Teardown.
  • Failures: Contains failures of the test's failed actions.
  • Failures.Setup: Contains failures from Setup.
  • Failures.EachSetup: Contains failures from EachSetup.
  • Failures.Test: Contains failures from Test.
  • Failures.EachTeardown: Contains failures from EachTeardown.
  • Failures.Teardown: Contains failures from Teardown.

Functions

Chronos.TestCaseResults:AddAction (state, action)
Adds an action to the Actions and Actions[ state ] arrays. Will mark the results as passed if they would otherwise remain pending.

Parameters:

  • state: The state during which the action occurred.
  • action: The action.
Chronos.TestCaseResults:AddFailure (state, failure)
Adds a failure to the Failures and Failures[ state ] arrays. Will mark the results as failed.

Parameters:

  • state: The state during which the failure occurred.
  • failure: The failure.
