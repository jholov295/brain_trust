Module Chronos.TestRunResults

Contains information about a test run.

Tables

TestRunResults

Functions

Chronos.TestRunResults.new (saveResults) Creates a new Chronos.TestRunResults.


Tables

TestRunResults


Fields:

  • StartTime: The start time of the test run.
  • EndTime: The end time of the test run.
  • Time: The total time of the test run.
  • Status: "Passed" "Pending" or "Failed"
  • Passed: True if the test run passed.
  • Pending: True if the test run is pending.
  • Failed: True if the test run failed.
  • TestSuites: An array of Chronos.TestSuiteResults. (Also indexed by name: TestSuites["My Test Suite"])
  • TestSuites.Passed: An array of Chronos.TestSuiteResults whose status is Passed.
  • TestSuites.Pending: An array of Chronos.TestSuiteResults whose status is Pending.
  • TestSuites.Failed: An array of Chronos.TestSuiteResults whose status is Failed.
  • TestCases: An array of Chronos.TestCaseResults. (Also indexed by name: TestCases["My Test Case"])
  • TestCases.Passed: An array of Chronos.TestCaseResults whose status is Passed.
  • TestCases.Pending: An array of Chronos.TestCaseResults whose status is Pending.
  • TestCases.Failed: An array of Chronos.TestCaseResults whose status is Failed.
