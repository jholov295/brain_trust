Module cllib.Debugging

Helper functions that can be used to help debug Lua scripts. It has some namespace and table dumping functions.

Author:

Ryan Pusztai

Release: 1.00 <09/05/2007>

Note:

    This module is part of cllib (Common Lua Library). You will need extra files to use these functions. See 'Usage:' section for more information.

Usage:

    To use these scripts all you need to do is require() at the top of the file.
    This is an example of using an included module: 
    require( "ModuleName" )
    This is an example of calling a function in that module: 
    ModuleName.CoolFunction( parameter1, parameter2 )

Functions

PrintNamespaceFunctionList (name, tbl)
Prints out a list of available functions and data in a specified namespace.

Parameters:

  • name: String of the name of the table tbl you are listing.
  • tbl: Table that you want to list all the functions and data.
TableDump (atable, prefix, tablelevel)
Recursively dumps the contents of a table.

Parameters:

  • atable: Table to dump.
  • prefix: [OPT] String to prefix to each item in the table.
  • tablelevel: Don't use it. Only used for the recursive call.
