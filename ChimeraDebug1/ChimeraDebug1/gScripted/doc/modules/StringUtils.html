StringUtils.EndsWith (s, suffix, caseSensitive)
Check if a string ends with a specified suffix. The check can be case sensitive or insensitive.

Parameters:

  • s: {string} String to be tested
  • suffix: {string} String to be checked as a possible suffix.
  • caseSensitive: [OPT] {bool} A flag indicating if the check should be performed in a case-sensitive way. This defaults to false.

Return value:

    {bool} true if the string specified by s ends with the string specified by suffix.
StringUtils.GetCRC16 (data, len)
Given a string of bytes, I return the CRC16 of the data. The algorithm used was adapted from an website claiming that this algorithm is the official CCITT CRC-16 algorithm. However, it seems that the claim to be the one true and correct CRC16 algorithm is controversial, as there are items the the specification that are ambiguous and open to multiple interpretations.

Parameters:

  • data: {string} The input data
  • len: [OPT] {int} The number of bytes of input data. If -1 is supplied for this parameter, the function determines the length. This defaults to -1.

Return value:

    {int} A 16-bit CRC value.
