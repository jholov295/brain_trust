Module gtx

This module is where you can find information that is gtx library wide. Like the build number.

Author:

Ryan Pusztai

Release: 1.00 <02/04/2008>

Tables

gtx.buildNumber This is the build number of the gtx library.


Tables

gtx.buildNumber
This is the build number of the gtx library. This can be used to make your script usable with many versions of the gtx library. This is available all the time so you don't have to have a gtx object to use it.

Example:

 if gtx.buildNumber < 2580 then
     -- Do something for any version lower than 2580
     error( "This script does not support version "..gtx.buildNumber.." of the gtx library." )
 end
