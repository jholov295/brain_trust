Module gScripted

gScripted is a Lua module that allows users to do many things for general purpose and Device testing. It allows users to write simple scripts in a language that is simple yet powerful. It uses Lua. Many functions are available from the gtx Library.

For a quick guide to Lua

Author:

Ryan Pusztai

Release: 1.00 <08/28/2008>

Note:

    When a file is loaded Lua will set the current working directory to the loaded scripts directory. Make sure your paths in your scripts are relative to the top level script (the script you are running).

Lua:

    To use gScripted you need to write scripts in Lua. You can find out more about Lua here:

  • Lua manual
  • Programming in Lua – First Edition (book)

Usage:

    lua [options] [script [args]]

    Options:

    • -e stat
        execute string 'stat'
    • -l name
        Require library 'name'
    • -v
        Show Lua version information.
    • --help
        Show Lua's help message. Look at this to see what other options are available./ul>

    General Information:

      Script parameters:
        When running a script you can access the command line parameters passed to the script (see above in the 'Usage' section) in a table called arg. The zero index always has the full file name of the running script. It also contains the rest of the parameters passed in as a normal Lua table. 
         print( "Script that is loaded: ", arg[0] )
         print( "Number of arguments sent to the script: ", #arg )
         print( "Unpacked arguments: ", unpack( arg ) )

    Parameters:

    • millis: {int} number of milliseconds to sleep.

    Tables

    gScripted.buildNumber
    This is the build number of gScripted. This can be used to make your script usable with many versions of gScripted. This number actually gives you finer control of what versions you can work with. This is available all the time so you don't have to have a gScripted object to use it.

    Example:

     if gScripted.buildNumber == 1054 then
         -- Do a finer grained version controls stuff here.
     end


    gScripted.version
    This is the version number of gScripted. This can be used to make your script usable with many versions of gScripted. This is available all the time so you don't have to have a gScripted object to use it.

    Example:

     if gScripted.version == 0.9 then
         -- Do 0.9 specific stuff
     end
