FileUtils.CopyFileTo (src, dest, overwriteIfExists)
Copy a file to another location. This function will fail if the destination path does not exist. Use CreateDirs() to implement this behavior.

Parameters:

  • src: {string} The source path and file name.
  • dest: {string} The destination path and file name.
  • overwriteIfExists: {bool} If true, the existing file will be overwritten.

Return value:

    {bool} false if copy failed, else true.
FileUtils.CreateDirs (path, skipLast)
Create all of the directories in the specified path. Optionally ignores the last element in the path if skipLast == true (do this for a file path).

Parameters:

  • path: {string} The complete path and file name of a file.
  • skipLast: [OPT] {bool} If true, ignores the last element in the path. Usefull for file paths only. This defaults to false.

FileUtils.GetBaseFileName (path)
Given a string which contains a filename or a path to a file, I return a string containing the same data passed in, with the file type suffix excluded. For example, if "c:\temp\foo.txt" is passed in, the result would be "c:\temp\foo".

Parameters:

  • path: {string} A string containing the input path to be parsed.

Return value:

    {string} The resulting base filename extracted from the path.

FileUtils.GetFileExtension (path)
Given a string which contains a path to file, I return a string containing the file extension. For example, if "c:\temp\foo.txt" is passed in, the result would be "txt".

FileUtils.GetFileName (path)
Given a string which contains a path to file, I return a string containing just the filename. For example, if "c:\temp\foo.txt" is passed in, the result would be "foo.txt".

Return value:

    {string} The resulting filename extracted from the path.

Make a file writable or read-only.
